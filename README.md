# 🍎 Qt Calculator (C++)

A modern Apple-style calculator built with Qt and C++.

## Features
- Basic operations (+ - × ÷)
- Clean Apple-inspired UI
- Dark mode design
- Responsive layout

## Build
```bash
mkdir build
cd build
cmake ..
make
./CalculatorQt
